• FM JS Engine is a 2D game engine entirely written in JavaScript that'll empower you to build your games quickly.
  • It's an easy to learn 2D game engine that should fit your every need while being customizable at the same time.
  • Games for everyone and for everywhere.
  • You don't need consoles, plugins or specific platforms.
  • You just make a game that'll run on every modern browsers.

If you want to check what you can do with it make sure to visit this page.
In case you are looking for information about the engine please go here.
You can also check the various games and experiments made with FM JS Engine here.
Do not hesitate to use the community forum for any questions and requests. And finally don't forget to download the engine !

The FM JS Engine comes in three flavors :

  • Full : if you need to change parts of the code for whatever good reasons.
  • One file : if you just need to be able to see parts of the code from time to time.
  • Minified : once you finished your game this is the best way to keep the file very small.
